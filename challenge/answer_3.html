By Huw

This is similar to the polyalphabetic cipher in cipher 2. Its difference, however, is that it uses the cipher for one block as the key for the next block. The first block is enciphered with ‘WALES’.

If you had the solution to cipher 2, you should have had no problem with cipher 3.
